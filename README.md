# Kavya Kanaja 📚

Kavya Kanaja is a modern Kannada poetry application developed using Kotlin and Jetpack Compose.  
The app helps users explore Kannada poems with explanations, meanings, favorites, and beautiful UI design.

---

## ✨ Features

- 📖 Browse Kannada poems
- 🧠 AI-based explanation
- 📚 Word meaning popup
- ❤️ Favorite poems
- 🔍 Search functionality
- ☁️ Firebase integration
- 🎨 Historical themed UI
- 📱 Modern Jetpack Compose design

---

## 🛠 Tech Stack

- Kotlin
- Jetpack Compose
- Firebase Firestore
- MVVM Architecture
- Android Studio

---

## 🚀 Installation

1. Clone the repository
2. Open in Android Studio
3. Sync Gradle
4. Connect Firebase
5. Run the app
